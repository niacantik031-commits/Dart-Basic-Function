Project ini adalah program sederhana menggunakan JavaScript yang dibuat untuk memahami dasar-dasar function. Di dalam project ini terdapat tiga topik utama, yaitu menyapa, menghitung luas persegi, dan menghitung diskon. Pada setiap topik, terdapat empat jenis function yang berbeda, yaitu function tanpa parameter tanpa return, function dengan parameter tanpa return, function tanpa parameter dengan return, dan function dengan parameter serta return.
Pada topik menyapa, program dibuat untuk menampilkan sapaan seperti memperkenalkan nama dan memberikan ucapan ulang tahun, misalnya dengan function sayMyName('Raraa') dan sayHappyBirthday('Raraa', 22). Kemudian pada topik kalkulator luas persegi, program menghitung luas dengan rumus sisi dikali sisi, sehingga pengguna bisa mengetahui hasil luas dari input yang diberikan. Terakhir, pada topik kalkulator diskon, program digunakan untuk menghitung harga akhir setelah mendapatkan potongan diskon berdasarkan persentase tertentu.
Setiap function dalam project ini juga dilengkapi dengan contoh pemanggilan serta komentar sederhana agar mudah dipahami, terutama bagi pemula yang sedang belajar JavaScript. Project ini dibuat sebagai latihan untuk memahami cara kerja function serta perbedaan tiap jenisnya secara dasar.
